About the Program

Journey is an innovative website designed to aid you in planning your perfect roadtrip. Just enter where you want to start, where you want to go, and how long you want to drive each day and Journey will do the rest for you!

Built using HERE's Map API, Geocoder API, Routes API, and Places API, Journey has the ability to map out the optimal route and provide a structured list of nearby restaurant and hotel options along the way.

Try it out and plan a road trip today!

Created by: Tianrui Chen and Adam Victor
Written in: Html, Css, Javascript
Additional resources used: Map API, Geocoder API, Routes API, Places API, Node.js, Express.js, jQuery/ajax
